# VocKO Nuxt Frontend

## Structure
- `pages/` — Main app pages (decks, learning session)
- `components/Flashcard.vue` — Flashcard UI

## Quickstart

```bash
cd web/nuxt
npx nuxi init .
npm install
npm run dev
```
